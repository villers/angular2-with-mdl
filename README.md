# Angular2Mdl

url: http://localhost:8080

Start dev server
```sh
$ npm start
```

Build production
```sh
$ npm run build
```

Clean directory
```sh
$ npm run clean
```

Install typings for angular
```sh
$ npm run typings-install
```

Install new typings (example jquery)
```sh
$ npm run typings-install-e jquery
```

Generate documentation
```sh
$ npm run docs
```
